# Sketch 002: Accent Placement

## Design Question

承接 sketch 001 winner(B: Pill-led)。底层结构定了——粉色去哪里?

3 档密度:粉只在主题名 / 粉在命令 + 主题 / 粉延伸到动作动词。

## How to View

```
open .planning/sketches/002-accent-placement/index.html
```

## Variants

- **A: Quiet** — 粉只在主题名 + ★。命令 pill 中性灰。粉是"奖励色",出现即 dopamine。
- **B: Medium** — 粉在命令 + 主题 + ★(sketch 001 winner 的原始配置)。粉是"功能识别色"。
- **C: Loud** — 粉再加动作动词(Run / Try / Switched / Cleaned)。最满,叙事感最强。

## What to Look For

- 5 个场景都和 sketch 001 完全相同——只比粉色密度。
- 看完场景 1 setup tail,你的眼睛被粉色"拉"了几次。
- 场景 4 error 行注意:粉(命令)和红(✗)同时出现是否打架。
- 哪个最像"slate 这个工具的人格"。
